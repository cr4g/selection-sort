How does Selection Sort work? (Simulation)

  • Find the smallest element in the list.
  • Swap it with the first element.
  • Move to the next position.
  • Repeat the process until the list is sorted.
SelectionSort(A, n) for i = 0 to n-2 minIndex = i for j = i+1 to n-1 if A[j] < A[minIndex] minIndex = j swap A[i] and A[minIndex]
